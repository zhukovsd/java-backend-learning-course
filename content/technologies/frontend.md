# Frontend

Рабочие задачи бэкенд разработчика часто косвенно касаются фронтенда - отдать по REST API данные, отобразить их на веб-интерфейсе. Поэтому, проекты данного роадмапа включают в себя проекты с веб-интерфейсом.

## Интеграция фронтенда и бэкенда

Что нужно знать:
- HTTP (GET, POST, cookies, headers)
- CORS
- REST
- Раздача фронтенд ресурсов бэкенд приложением

Практика - проекты с #3 по #7.
